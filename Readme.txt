Manta Property Extension is a simple extension for Windows Explorer that allows users (who are most probably developers) to see whether a Microsoft Portable Executable file like EXE and DLL are 32/64-bit architecture targeted, run in CUI/GUI mode and others.

For more information, please visit: http://sanje2v.wordpress.com/2013/12/10/writing-property-handler-for-windows-explorermanta-property-extension/


Setup for binary is available in 'Setup' folder inside this repository's project folder.
